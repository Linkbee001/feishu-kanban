New endpoint per RESEARCH.md:
```typescript
@Get('api/admin/groups')
listGroups(
  @Query('status') status?: string,
  @Query('search') search?: string,
  @Query('page') page?: string,
  @Query('limit') limit?: string,
) {
  return this.admin.listGroups({
    status,
    search,
    page: page ? parseInt(page, 10) : 1,
    limit: limit ? parseInt(limit, 10) : 25,
  });
}
```

AdminService.listGroups implementation:
- Query GroupAgentSession with project relation
- Map sessionMode to status: active, pending_config, bootstrap
- Join with Project for project names
- Support filtering by status and search (name/chatId)
- Return paginated results with total count

Response shape:
